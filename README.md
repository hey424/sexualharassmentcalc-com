# sexualharassmentcalc-com

Astro + Cloudflare Workers static site for sexualharassmentcalc.com, an educational resource and informational settlement calculator for workplace sexual harassment claims under Title VII of the Civil Rights Act of 1964 and state equivalents (FEHA, NYHRL, and others). Editorial direction by our editorial team (EEOC Practice & Title VII, SHRM-SCP and employment-law paralegal expertise, plus State Harassment Statutes & Procedural Reference paralegal expertise). Live pages use a generic "Editorial Team" byline, no named personas. The site offers a non-binding range estimator, a methodology page documenting damages components and Title VII employer-size caps, deep guides on claim types, procedural steps, and common misconceptions, plus standard privacy, terms, contact, FAQ, and editorial pages. Build with `npm run build`, preview with `npm run preview`, and deploy the `dist/` directory via `wrangler deploy`.
