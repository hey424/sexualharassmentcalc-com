# sexualharassmentcalc-com

Astro + Cloudflare Workers static site for sexualharassmentcalc.com, an educational resource and informational settlement calculator for workplace sexual harassment claims under Title VII of the Civil Rights Act of 1964 and state equivalents (FEHA, NYHRL, and others). Editorial direction by Devin Bertram (DB), EEOC Practice & Title VII Lead, SHRM-SCP and employment-law paralegal, with contributing reference work from Harlow Pace (HP), State Harassment Statutes & Procedural Reference paralegal. The site offers a non-binding range estimator, a methodology page documenting damages components and Title VII employer-size caps, deep guides on claim types, procedural steps, and common misconceptions, plus standard privacy, terms, contact, FAQ, and editorial pages. Build with `npm run build`, preview with `npm run preview`, and deploy the `dist/` directory via `wrangler deploy`.
